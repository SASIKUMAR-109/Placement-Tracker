📊 About This Project

This is a portfolio project demonstrating a full-stack data pipeline — Python scraper, SQLite database, Flask REST API, and interactive dashboard.

Data is sourced from NIRF government rankings (2017–2025). Some placement figures may be inaccurate due to inconsistencies in the source dataset. The pipeline and visualisation are fully functional.
